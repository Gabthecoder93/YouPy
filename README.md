YouPy

🎥 YouPy est un clone minimaliste de YouTube développé en Python. Il permet à un utilisateur de :

   S’inscrire et se connecter

  Envoyer une vidéo depuis son PC
    Voir les vidéos des autres
    Liker, commenter, s’abonner
    Naviguer entre les profils
    Et bien plus encore…
  
    

📦 Télécharger l’application

🟢 Télécharger la dernière version (.exe) sur la page des Releases

Contenu du dossier à télécharger :

  You.exe → l'application principale
    ffmpeg.exe → pour convertir les vidéos
    ffprobe.exe → pour analyser les métadonnées des vidéos

📝 Tous ces fichiers doivent être placés dans le même dossier pour que l'application fonctionne.

ℹ️ Compatible Windows uniquement.
🧠 Technologies utilisées

  Python 3.12

   tkinter pour l'interface

  Supabase (base de données et authentification)

  ffmpeg / ffprobe pour la conversion vidéo

  PyInstaller pour créer l'exécutable

⚙️ Instructions pour les développeurs

Pour exécuter YouPy depuis le code source :

  Cloner le repo :

git clone https://github.com/Gabthecoder93/YouPy.git
cd YouPy

  Installer les dépendances :

pip install -r requirements.txt

 Placer ffmpeg.exe et ffprobe.exe dans le même dossier que le script principal
 Lancer le programme :

python You.py

✨ Remerciements

  ffmpeg – pour leur outil open-source puissant
